## HOW TO LAUNCH
1. Clone the REPO in terminal
* ```git clone git@github.com:bcchen52/p00_story.git```
2. CD into the REPO
* ```cd p00_story```
3. Install Requirements
* ```pip freeze > requirements.txt```
* ```pip install requirements.txt```
4. Run the Flask Server
* ```cd app```
* ```python3 __init__.py```
